        An Architecture for a Secure Cloud Collaboration System
               draft-jennings-perpass-secure-rai-cloud-00

Abstract

   This draft outlines an architecture for building cloud collaboration
   systems where the users retain possession of the keys to decrypt the
   data and the cloud servers never see the keys.

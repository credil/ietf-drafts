        RFC 3430
                                      
        Title:      Simple Network Management Protocol Over
                    Transmission Control Protocol Transport Mapping
        Author(s):  J. Schoenwaelder
        Status:     Experimental
        Date:       December 2002
        Mailbox:    schoenw@ibr.cs.tu-bs.de
        Pages:      10
        Characters: 21527
        Updates/Obsoletes/SeeAlso:    None
                                      
        I-D Tag:    draft-irtf-nmrg-snmp-tcp-09.txt

        URL:        ftp://ftp.rfc-editor.org/in-notes/rfc3430.txt


This memo defines a transport mapping for using the Simple Network
Management Protocol (SNMP) over TCP.  The transport mapping can be
used with any version of SNMP.  This document extends the transport
mappings defined in STD 62, RFC 3417.

This memo defines an Experimental Protocol for the Internet community.
It does not specify an Internet standard of any kind.  Discussion and
suggestions for improvement are requested.  Distribution of this memo
is unlimited.

        RFC 3147

        Title:      Generic Routing Encapsulation over CLNS Networks 
        Author(s):  P. Christian
        Status:     Informational
        Date:       July 2001
        Mailbox:    christi@nortelnetworks.com
        Pages:      8
        Characters: 14125
        Updates/Obsoletes/SeeAlso:  None

        I-D Tag:    draft-christian-gre-over-clnp-02.txt

        URL:        ftp://ftp.rfc-editor.org/in-notes/rfc3147.txt


This document proposes a method for transporting an arbitrary protocol
over a CLNS (Connectionless Network Service) network using GRE
(Generic Routing Encapsulation).  This may then be used as a method to
tunnel IPv4 or IPv6 over CLNS.

This memo provides information for the Internet community.  It does
not specify an Internet standard of any kind.  Distribution of this
memo is unlimited.

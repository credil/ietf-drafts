       Verification Involving PSTN Reachability (VIPR): Framework
                 draft-petithuguenin-vipr-framework-00

Abstract

   This document describes the framework for Verification Involving PSTN
   Reachability (VIPR), a set of protocols to automatically and securely
   provision SIP routes between domains.
